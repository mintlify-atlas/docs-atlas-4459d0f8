Compose Svelted is an open-source UI library for Svelte that maps the **Jetpack Compose mental model** to the web. If you've built Android UIs with Compose, the components, modifiers, alignment tokens, and navigation patterns will feel immediately familiar — without any virtual DOM abstraction or Skia dependency.

## Get started in 3 steps

<Steps>
  <Step title="Install the package">
    ```bash
    npm install @danielito1996/compose-svelted
    ```
  </Step>
  <Step title="Import the CSS baseline">
    ```ts
    import "@danielito1996/compose-svelted/baseline.css";
    ```
  </Step>
  <Step title="Wrap your app and build">
    ```svelte
    <script>
      import { ComposeTheme, AppRoot, Column, Text } from "@danielito1996/compose-svelted";
    </script>

    <ComposeTheme mode="system">
      <AppRoot>
        <Column>
          <Text textStyle="headlineMedium">Hello, Compose Svelted!</Text>
        </Column>
      </AppRoot>
    </ComposeTheme>
    ```
  </Step>
</Steps>

<Note>
  Compose Svelted requires **Svelte 5** as a peer dependency. It is currently in **alpha** — the core API (layouts, modifiers, theme, motion, navigation) is stable, with more features planned in Core V3.
</Note>
